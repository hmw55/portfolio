<h2 align=center style="color: #32d583">
Local Development
</h2>

<h3 align=center>
Clone Repository
</h3>

#### SSH
```bash
git clone git@github.com:hmw55/portfolio.git
```

#### HTTPS
```bash
git clone https://github.com/hmw55/portfolio.git
```
```bash
cd portfolio
```

<h3 align=center>
Install Dependencies
</h3>

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

<h2 align=center  style="color: #32d583">
Environment Variables
</h2>

Create a `.env.local` file:

```text
VITE_TURNSTILE_SITE_KEY=your_turnstile_site_key
```

Production environment variables are configured in Vercel:

RESEND_API_KEY=

TURNSTILE_SECRET_KEY=

VITE_TURNSTILE_SITE_KEY=
